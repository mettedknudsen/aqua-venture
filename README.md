## Setup

Make sure to install dependencies:

```bash
# node version
v24.14.0

# pnpm
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# pnpm
pnpm run dev 

or

nuxt dev
```

## Production

Build the application for production:
```bash
# pnpm
pnpm build
```

Locally preview production build:

```bash
node .output/server/index.mjs    
```
